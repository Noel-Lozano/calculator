# Calc

A simple Python calculator class that supports basic arithmetic operations.

## Features

- Addition (`add`)
- Subtraction (`sub`)
- Multiplication (`mul`)
- Division (`div`)
- Exponentiation (`pow`)
- Square Root (`sqrt`)

## Usage

```python
from calc import Calc

c = Calc()
print(c.add(5, 3))     # Output: 8
print(c.sqrt(16))      # Output: 4.0
